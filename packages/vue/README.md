# @sdkforge/vue

The official native Vue 3 adapter plugin for **SDKForge**. 

Automatically handles reactivity natively building composable mappings tied to perfectly constructed Clean Architecture hooks securely typed.

## Installation
```bash
npm install -D @sdkforge/cli @sdkforge/vue
```

## Usage
```bash
npx @sdkforge/cli generate -i api.json -o src/api --plugin vue
```
