# @contractforge/vue

The official native Vue 3 adapter plugin for **ContractForge**. 

Automatically handles reactivity natively building composable mappings tied to perfectly constructed Clean Architecture hooks securely typed.

## Installation
```bash
npm install -D @contractforge/cli @contractforge/vue
```

## Usage
```bash
npx @contractforge/cli generate -i api.json -o src/api --plugin vue
```
