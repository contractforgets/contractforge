# @contractforge/nest

The official NestJS framework adapter plugin for **ContractForge**. 

Maps standard schema DTO interfaces directly into Class-validator NestJS controller bindings reducing enormous boilerplate instantly!

## Installation
```bash
npm install -D @contractforge/cli @contractforge/nest
```

## Usage
```bash
npx @contractforge/cli generate -i api.json -o src/api --plugin nest
```
